üîß How to Update Your Logo

Step 1: Replace the placeholder files

Step 2: Update HTML files

Step 3: Replace the logo code

Find this code in each HTML file:

<div class="w-9 h-9 rounded-md bg-[#064b67] grid place-items-center text-white font-bold">DV</div>

Replace it with:

<img src="./assets/images/logos/logo.png" alt="Divine Vision Hardware" class="h-9 w-auto" />

Step 4: Update favicon

Add this line in the <head> section of each HTML file:

<link rel="icon" type="image/x-icon" href="./assets/images/logos/favicon.ico">
